/tp

Телепортация

Основные команды перемещения по серверу: спавн, случайная телепортация, запросы TPA и возврат назад.

/spawn
Телепортация на точку спавна сервера.
tanuki.teleport.spawn
/setspawn
OP
Установить точку спавна сервера на текущую позицию.
tanuki.admin.setspawn
/back
Вернуться на предыдущую позицию (до последней телепортации).
tanuki.teleport.back
/tpr
алиасы: rtp wild
Случайная телепортация в безопасное место. Система избегает лаву, кактусы, магму и огонь. Поиск безопасной точки происходит асинхронно.
tanuki.teleport.rtp Кулдаун: 300 сек.
/tpa <игрок>
Отправить запрос на телепортацию к указанному игроку. Игроки с правом tanuki.teleport.tpbypass телепортируются мгновенно.
tanuki.teleport.tpa имя игрока
/tpahere <игрок>
Отправить запрос игроку, чтобы он телепортировался к вам.
tanuki.teleport.tpa имя игрока
/tpaccept
Принять входящий запрос на телепортацию.
tanuki.teleport.tpa
/tpdeny
Отклонить входящий запрос на телепортацию.
tanuki.teleport.tpa
/h

Дома

Управление домашними точками телепортации. По умолчанию каждый игрок может сохранить до 3 домов.

/home [название]
Телепортация к сохранённому дому. Если название не указано, телепортирует к дому home.
tanuki.teleport.home название дома
/sethome [название]
Сохранить текущую позицию как дом. Если название не указано, используется home. Можно обновить существующую точку.
tanuki.teleport.home название дома
/delhome <название>
Удалить сохранённый дом.
tanuki.teleport.home название дома
/homes
Показать список всех сохранённых домов с количеством и лимитом. Интерактивный список — нажмите на название, чтобы телепортироваться.
tanuki.teleport.home
/w

Варпы

Серверные точки телепортации, созданные администрацией. Доступны для всех игроков.

/warp <название>
Телепортироваться на указанный серверный варп.
tanuki.teleport.warp название варпа
/setwarp <название>
OP
Создать новый серверный варп на текущей позиции. Нельзя создать варп с уже существующим названием.
tanuki.admin.setwarp название варпа
/delwarp <название>
OP
Удалить серверный варп.
tanuki.admin.delwarp название варпа
/warps
Показать список всех доступных варпов. Интерактивный список — нажмите на название, чтобы телепортироваться.
tanuki.teleport.warp
/c

Регионы

Система приватных территорий. Выделите зону палкой (2 точки) и создайте регион. По умолчанию можно создать до 5 регионов. Поддержка Java и Bedrock Edition.

/claim
Создать регион по двум выделенным точкам или открыть меню управления текущим регионом. Система проверяет минимальный и максимальный размер, а также пересечения с другими регионами.
tanuki.claim.create
/claims
Показать список всех ваших регионов. Интерфейс адаптирован для Java и Bedrock Edition.
tanuki.claim.create
/unclaim
Удалить регион, в котором вы находитесь. Только владелец или администратор может удалить регион.
tanuki.claim.create
/claiminfo
Показать информацию о текущем регионе: владелец, мир, координаты, площадь, количество доверенных игроков.
tanuki.claim.create
/trust <игрок>
Добавить игрока в список доверенных текущего региона. Только владелец или администратор может управлять доверенными.
tanuki.claim.trust имя игрока
/untrust <игрок>
Убрать игрока из списка доверенных текущего региона.
tanuki.claim.trust имя игрока
/trustlist
Показать список всех доверенных игроков в текущем регионе. Интерфейс адаптирован для Java и Bedrock Edition.
tanuki.claim.trust
/te

Утилиты

Дополнительные команды: управление HUD и справочная система плагина.

/tanukihud [on | off]
алиасы: thud bossbar
Переключить отображение HUD (BossBar). Если аргумент не указан, переключает текущее состояние. Настройка сохраняется для каждого игрока.
tanuki.bossbar.toggle on / off
/tehelp [категория]
алиасы: tanukihelp tecommands te
Внутриигровая справка по командам плагина. Можно указать категорию для фильтрации: teleport, homes, warps, claims или all.
tanuki.help категория
